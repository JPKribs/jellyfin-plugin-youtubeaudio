Troubleshooting

Reset individual sync tables or the entire database. These actions are destructive and cannot be undone.

Reset Individual Tables

Content Sync
Remove all content tracking data. You will need to re-scan content.
History Sync
Remove all history tracking data. You will need to re-scan history.
Metadata Sync
Remove all metadata tracking data. You will need to re-scan metadata.
User Sync
Remove all user sync tracking data. You will need to re-scan users.

Reset Entire Database

Full Database Reset
Delete the entire sync database file and recreate it from scratch. All tracking data across all sync types will be lost.
